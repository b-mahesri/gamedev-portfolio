Web Systems Coursework

My upper level web systems course had me and 2 of my teammates building several medium sized full-stack projects. The most notable were the following:

Social Media Web App Clone

What we did:

  • Built an Instagram clone with client-side and server-side rendering
  • Backend: Designed and implemented RESTful API endpoints using Python’s Flask, integrated with a SQLite database
  • Frontend: Created with JavaScript React components and HTML / CSS templates

Tech Stack and Project Management tools:

SQL, Python, Flask, JavaScript, ReactJS, Bash, Github, AWS, VSCode

MapReduce Framework

What we did:

  • Designed and implemented a MapReduce framework in Python inspired by Google’s original MapReduce paper
  • Built Manager and Worker components communicating via TCP/UDP sockets, and used multithreading and OS concurrency tools to handle parallel map and reduce operations

Tech Stack and Project Management tools:

Python, Bash, Github, VSCode

Overall

Our Process:

All projects required we develop our conceptual and technical proficiency on the fly. Our approach was to learn together. We did all our coding together, 3 programmers, one editor. We helped eachother through every step of the projects, from learning JavaScript and React to understanding how to structure a MapReduce framework.

My Contributions

I enjoyed taking initiative in the moments when the team was stuck, and taking the time to study the problem independently before team meetings and then walking through my thinking to help get the ball rolling for everyone.
